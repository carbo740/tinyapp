## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- cookie-session

# Getting Started

- Install all dependencies using 'npm install "dependency"'.
- Run the main web server file by using 'node express_server.js'.
- Register first to access all functionalites.
- Make sure to use "http" for your long links.
